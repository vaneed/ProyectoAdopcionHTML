## 🚩 Problemática

Muchas fundaciones de rescate animal realizan una labor de buen corazón, pero no cuentan con una presencia en redes o en la web que sea clara y atractiva hacia un público remoto que les permita:

- 🐶 Dar a conocer su trabajo  
- 🏠 Mostrar animales disponibles para adopción  
- 🤝 Facilitar el contacto con posibles adoptantes y voluntarios  
- 💵 Recibir apoyo mediante donaciones  

Esto limita su alcance y reduce las oportunidades de ayuda para los animales y el crecimiento de la fundación.

---

## 💡 Solución

Se desarrolló una maqueta web estática para la **Fundación Patitas Felices**, diseñada para presentar de forma clara e interactiva (usando solo CSS) toda la información relevante.

El sistema permite visualizar:

- ✅ Información de la fundación (misión y visión)  
- ✅ Animales disponibles para adopción  
- ✅ Perfiles detallados de cada animal  
- ✅ Formulario de contacto para distintos tipos de consulta  

---

## 🔍 Detalles principales

- 📱💻 Diseño responsive adaptable a móviles, tablets y escritorio  
- 🧱 Uso de Flexbox y CSS Grid para la estructura  
- 🎞️ Carrusel automático de imágenes usando `@keyframes` (sin JavaScript)  
- 🎯 Formularios mediante CSS  
- 🐾 Interfaz visual amigable y enfocada en la experiencia del usuario

## 🧩 Funcionalidades implementadas

### 🏠 Página de inicio
- Banner principal con nombre y logo  
- Carrusel automático de imágenes  
- Secciones destacadas con enlaces internos  

### 📖 Misión y visión
- Diseño en columnas (PC) y apilado (móvil)  
- Contenido claro y estructurado  

### 🖼️ Galería
- Mosaico o carrusel de imágenes de rescates y actividades  

### 🐕 Adopta
- Grid de tarjetas con animales  
- Información básica (nombre, edad, estado de salud)  
- Botón **“Ver perfil”**  

### 📋 Perfil de animal
- Imagen principal  
- Historia y características  
- Botón de solicitud de adopción  

### 📞 Contacto
- Formulario con campos obligatorios  
- Validaciones visuales con CSS (`.error`, `.success`)  
- Información adicional (correo, teléfono, redes)  

### 🧾 Footer
- Nombre de la fundación  
- Información de contacto
